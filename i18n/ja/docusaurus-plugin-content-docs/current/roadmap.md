MapConductor は、まだ初期段階にありながらも活発に開発が進んでいるプロジェクトです。短期的な優先事項は、Android SDK を実用レベルまで安定させることと、初心者でも扱いやすいシンプルな API を保つことです。

## 短期的な目標（進行中）

- Android SDK を堅牢化し、開発体験を向上させる。
- Google Maps、Mapbox、ArcGIS、HERE、MapLibre への対応を拡充・改善する。
- 一般的な位置情報アプリケーション向けの、わかりやすいチュートリアルやサンプルを整備する。

## 中期的な目標

- 同様の統一 API を持つ iOS SDK のプロトタイプを開発する。
- 追加の地図オーバーレイやインタラクティブな機能を検討する。
- 特定のプロバイダに依存しない形で、データ同期やジオフェンシングのためのクラウド連携コンセプトを設計する。

## コミュニティと資金

このプロジェクトは、地図アプリ開発の実体験に根ざした個人の取り組みとして始まり、現在は日本の独立行政法人情報処理推進機構（IPA）が運営する「未踏アドバンスト（Mitou Advanced）」プログラムの支援を受けています。

今後は、MapConductor をモバイルエコシステムにおける公共のオープンソース基盤として成長させるため、NLnet の Mobifree プログラムに応募しています。

Android SDK はすでにリリースされており、デモアプリケーションでも利用されています。次のバージョンも近日中のリリースを予定しています。Mobifree の支援期間中には、Android SDK のさらなる安定化に加え、iOS SDK（可能であれば Flutter SDK も）の開発を進め、マーカークラスタリングやラスタ／ベクタタイルのレンダリングなどの高度な機能にも取り組む計画です。

また、資金は開発時間の確保だけでなく、世界各地での発表やイベント参加、バグハンティングや貢献度の高いコントリビューターへの謝礼など、コミュニティ活動の強化にも活用したいと考えています。

MapConductor を持続可能なコミュニティプロジェクトとして育てることを重視しています。そのために、明確なガバナンス、透明性のある意思決定、多様な人々からの貢献を歓迎する環境づくりを進めていきます。
